# Shell Toolbox

A simple, modular, and highly configurable toolbox. Designed to be easily integrated in containers

## Structure

```bash
shell_toolbox/
├── install.sh          
├── toolbox.sh          
├── modules/            
│   ├── prompt.sh       
│   ├── git.sh          
│   ├── aliases.sh      
│   └── utils.sh        
├── config.env          
└── README.md    
```

## Installation

Clone and install with one command:

```bash
git clone <your-repo-url> ~/.shell_toolbox
bash ~/.shell_toolbox/install.sh
source ~/.bashrc
```
